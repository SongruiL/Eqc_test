# 级联计算演示 (CASCADE)

**模型**: CascadeDemo
**描述**: 展示方程之间的 DAG 依赖关系

## 参数

| 名称 | 中文名 | 默认值 | 单位 | 可优化 |
|------|--------|--------|------|--------|
| p1 | 一阶系数 | 0.5 | - | 是 |
| p2 | 二阶系数 | 0.3 | - | 是 |

## 变量

| 名称 | 类型 | 单位 | 描述 |
|------|------|------|------|
| x | Input | - | 原始输入 |

## 方程

### stage1 - 一阶变换

**输出**: `y1`

**公式**: $p1 \times x$

**依赖**: x, p1

### stage2 - 二阶变换

**输出**: `y2`

**公式**: $y1 + p2 \times x$

**依赖**: y1, x, p2

### stage3 - 最终输出

**输出**: `y_final`

**公式**: $\sqrt{|y2|}$

**依赖**: y2
